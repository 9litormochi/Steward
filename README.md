## Useage
+ `on`/`off`/`del` 启动/禁用/删除相应的扩展
+ `run` 命令启动App
+ `tab` 命令查找并定位到打开的标签页
+ `his` 命令查找历史记录并新标签打开
+ `yd` 有道词典
+ `todo` 待办事项，标签页显示
+ `po` 命令查找pocket
