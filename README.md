## Useage
+ `on`/`off` + `space` 显示所有**未启用** / **启用中**的扩展
+ `on`/`off` + text 查找匹配相应的扩展
+ `enter` 回车执行操作
+ `tab` 命令查找并定位到打开的标签页
+ `his` 命令查找历史记录并新标签打开
+ 支持简写匹配，比如：`kzcx` 可以匹配 `扩展程序`
